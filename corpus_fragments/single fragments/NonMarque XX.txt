XX NON MARQUÉ
Nous vivons sous trois codes, le code naturel, le code civil, le code religieux. Il est évident que tant que ces trois sortes de législations seront contradictoires entre elles, il est impossible qu'on soit vertueux. Il faudra tantôt fouler aux pieds la nature, pour obéir aux institutions sociales, & les institutions sociales, pour se conformer aux préceptes de la religion. Qu'en arriverat-il? C'est qu'alternativement infracteurs de ces différentes autorités, nous n'en respecterons aucune; & que nous ne serons ni hommes, ni citoyens, ni pieux.
